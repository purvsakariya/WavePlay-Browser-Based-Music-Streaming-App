# 🎵 WavePlay – Modern Web Music Player

### WavePlay is a fully functional browser-based music streaming application inspired by Spotify.This project demonstrates real-world JavaScript concepts including dynamic song loading, playlist management, and audio control handling.

## 🚀 Features

### 🎶 Play / Pause functionality
### ⏭ Next & Previous song navigation
### 📂 Multiple playlists support
### 📊 Interactive progress bar
### 🔊 Volume control
### 💾 Local song management
### 📱 Fully responsive UI
### 🎨 Clean and modern interface

## 🛠️ Tech Stack

### HTML5
### CSS3
### JavaScript (Vanilla JS)

## 🧠 Concepts Implemented

### DOM Manipulation
### Event Handling
### State Management
### Array & Object Handling
### Dynamic Rendering
### Audio API Integration
### Modular Function Structure
